# Custom Layouts Sample
While the VidyoConnector provides a default layout, there is also a means to customize it to meet the needs of your UI/UX. The two modes of operation, which are mutually exclusive: Composie and Custom Layouts. This sample demonstrate how the custom layout option could be integrated to display each remote participant seperately.

Lean more: https://developer.vidyo.io

# Clone
git clone https://github.com/Vidyo/VidyoConnector-Android-CustomLayouts.git
